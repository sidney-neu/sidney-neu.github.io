云原生的起源

Docker的诞生

2008年dotCloud公司成立，该公司专注于PaaS平台为开发者或研发厂商提供技术服务。2013年Docker作为公共项目发布，同年dotCloud改名为Docker Inc，转型专注于Docker引擎和Docker生态系统。

在Docker的官网上有以下的介绍：

Docker takes away repetitive, mundane configuration tasks and is used throughout the development lifecycle for fast, easy and portable application development – desktop and cloud. Docker’s comprehensive end to end platform includes UIs, CLIs, APIs and security that are engineered to work together across the entire application delivery lifecycle.

关于Docker的其他介绍可以参见笔者的另外一篇博客：云原生简介。

Docker的操作系统支持

Docker支持在多种操作系统上的安装和运行。

Docker类型 操作系统 安装要求
Docker Server Linux X86架构：CentOS、Debian、Fedora、Ubuntu
ARM 64架构：CentOS、Debian、Fedora、Ubuntu
ARM 32架构：Debian、Raspbian、Ubuntu
Kernel版本>=3.10
iptables版本>=1.4
Docker Desktop Windows Windows10 64-bit家庭/专业版 2004 build19041
windows10 64-bit企业/教育版 1909 build18363
WSL2或Hyper-V后台模式
Docker Desktop Mac macOS版本大于10.14
不能同时安装VirtualBox版本4.3.30以上
4G以上内存
Apple自研芯片版本：加装Rosetta 2
